# assets

## Ảnh nền cho mode Wibu 🌸

Để dùng ảnh nền riêng cho **mode Wibu**:

1. Đặt một ảnh tên `wibu-bg.jpg` (hoặc `.png` / `.webp` / `.gif`) vào thư mục này.
2. Chạy lại: `python3 build_blog.py`
3. Commit + push.

Build sẽ tự phát hiện ảnh và gắn vào nền mode Wibu (cho cả bản VI và EN).

Không có ảnh thì mode Wibu dùng **nền gradient tím–hồng mặc định** — vẫn đẹp, vẫn dùng được.

Mẹo: ảnh ngang, độ phân giải cao (vd 1920×1080+), tông tối một chút để chữ trên panel mờ kính dễ đọc.
